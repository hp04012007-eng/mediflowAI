# MediFlow AI

AI-Powered Hospital Intelligence Platform

## Overview

MediFlow AI is a healthcare management platform that leverages Google Gemini AI to improve patient triage, reduce waiting times, optimize hospital queues, and provide administrators with real-time operational insights.

## Features

* AI Symptom Analysis
* Automated Triage System
* Smart Queue Management
* Patient Dashboard
* Doctor Dashboard
* Admin Command Center
* Appointment Scheduling
* Analytics Dashboard
* Gemini Resource Recommendations
